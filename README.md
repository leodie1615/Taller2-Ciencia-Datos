Para ejecutar el notebook taller1_final.ipynb se debe ejecutar en consola cmd el comando ".\setup.bat", con el fin de instalar ambiente virtual y librerias necesarias para la ejecucion del notebook. Finalemente, se debe seleccionar el kernel venv para ejecutar las celdas de código.

# Objetivo del taller:

Aplicar técnicas estadísticas y de visualización para analizar el dataset de reservas hoteleras, con el fin de identificar patrones de demanda, factores asociados a la ocupación, la duración de la estadía y la probabilidad de cancelación; y, a partir de estos hallazgos, proponer recomendaciones accionables para optimizar políticas de precios, gestión de cancelaciones y anticipación de reservas en la cadena hotelera.

# Objetivos específicos:

- Estrategia de análisis.
- Explorar y depurar los datos para asegurar calidad y consistencia.
- Analisis univiariado de las variables seleccionadas.
- Validación perfil cliente que no cancelo / cliente que cancelo.
- Hipótesis.
